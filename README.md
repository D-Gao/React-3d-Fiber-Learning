## Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/D-Gao/React-3d-Fiber-Learning.git
   ```
2. Navigate to the project directory:
   ```sh
   cd to the project folder
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

## Usage

To run the project locally, use the following command:

```sh
npm run dev
```
